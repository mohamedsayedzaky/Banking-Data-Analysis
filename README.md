  <section>
            <p>This project provides an in-depth analysis of banking system data to drive operational efficiency, enhance customer experience, and optimize decision-making processes. The data analysis pipeline leverages cutting-edge tools and platforms such as Azure Synapse Analytics, SQL Server, Azure Data Factory, and Power BI.</p>

  <h2>Technologies Used</h2>
            <ul>
                <li><strong>SQL Server</strong> - Data storage and management</li>
                <li><strong>Azure Data Factory</strong> - ETL (Extract, Transform, Load) processes</li>
                <li><strong>Azure Synapse Analytics</strong> - Data analysis and processing</li>
                <li><strong>Power BI</strong> - Interactive visualizations and reporting</li>
                <li><strong>Python</strong> - Data manipulation and automation</li>
            </ul>

  <h2>Project Overview</h2>
            <p>The primary objective of this project is to extract, clean, and analyze banking data to derive actionable insights. The data is processed using Azure Data Factory and Azure Synapse Analytics, which ensures scalability and reliability. Power BI is then used to generate dynamic reports and dashboards for improved decision-making. The project also emphasizes data security and privacy, ensuring compliance with industry standards.</p>

  <h2>Key Features</h2>
            <ul>
                <li><strong>Data Transformation & ETL:</strong> Leveraged Azure Data Factory for seamless data extraction and transformation.</li>
                <li><strong>Advanced Analytics:</strong> Performed in-depth analysis using Azure Synapse Analytics for high-performance processing.</li>
                <li><strong>Interactive Dashboards:</strong> Utilized Power BI to create actionable reports and dashboards that aid business decisions.</li>
                <li><strong>Data Security:</strong> Prioritized secure data handling practices throughout the project lifecycle.</li>
                <li><strong>Automation:</strong> Automated data pipelines and reporting processes to enhance efficiency and reduce human error.</li>
            </ul>
